# ポモドーロ×Todo アプリ 要件定義書

## 概要

ポモドーロテクニックとタスク管理を組み合わせた、シンプルな作業効率化アプリ。

## 技術スタック

| 項目           | 採用                              |
| -------------- | --------------------------------- |
| フレームワーク | Next.js 16 (App Router)           |
| 言語           | TypeScript                        |
| スタイル       | Tailwind CSS v4                   |
| 状態管理       | React state (useState/useReducer) |
| 永続化         | localStorage                      |
| アイコン       | Next.js image icons / SVG inline  |

## 機能要件

### 1. ポモドーロタイマー

- 標準: **25分 作業** → **5分 休憩** のサイクル
- 再生/一時停止/リセット/スキップ操作
- カウントダウン表示 (MM:SS フォーマット)
- 残り時間が10秒になると視覚で通知（色の遷移など）
- サウンド通知（任意、将来拡張用）

### 2. Todo リスト

- タスクの追加・削除・完了チェック
- タスク名のみ（説明・期日なし、シンプルに）
- 完了/未完了で表示を切り替え
- 全タスク表示 / 未完了のみの切り替え（デフォルト）

### 3. タイマー × Todo の連携

- タイマー計画中に「現在取り組むタスク」を選択可能
- 進行中のタスクがタイマー表示上部に表示される
- タスク完了と同時に1回ポモドーロを記録

### 4. データ永続化 (localStorage)

| 保存内容       | キー名           | 形式                                               |
| -------------- | ---------------- | -------------------------------------------------- |
| Todo リスト    | `pomodoro-todos` | Todo[]                                             |
| タイマー状態   | `pomodoro-timer` | { timeLeft, isRunning, phase, completedPomodoros } |
| 累計ポモドーロ | `pomodoro-stats` | { todayCount, weekCount }                          |

## コンポーネント構成

```
app/page.tsx (トップレベル状態管理)
├── Header.tsx            (タイトル)
├── TimerSection.tsx      (タイマー + コントロール)
├── ProgressSection.tsx   (ポモドーロ進捗)
├── TodoSection.tsx       (Todo リスト)
└── TodoInput.tsx         (タスク追加フォーム)
```
